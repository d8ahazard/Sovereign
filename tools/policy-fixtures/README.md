# tools/policy-fixtures

Reusable fixtures and sample desired-state declarations for exercising the policy engine
([`agent_start.md`](../../agent_start.md) sections 8 and 12). These support deterministic,
repeatable policy tests (detection, plan, apply, verify, repair, rollback) without depending
on a live system.

Will contain harmless sample policies and recorded state snapshots used by unit and
integration tests.

_Placeholder until the declarative policy engine is implemented (Milestone 2)._
